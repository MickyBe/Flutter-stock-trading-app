# Shopping app
This is a simple shopping app used for teaching purposes for the [GDG Flutter Tech Camp](https://gdgaddis.gitbook.io/mobile-development-reference-tce-c01/). Uses the [Fake Store API](https://fakestoreapi.com/) for getting a list of product items and has features like adding to cart and showing items in a cart.
